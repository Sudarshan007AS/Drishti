About the Project

Drishti is a streamlined Dynamic Portfolio Dashboard designed to present investment data in a clear, organized, and professional format. The focus of the project is on structuring portfolio information, computing essential financial metrics, and offering an interface that helps users quickly understand the overall performance of their holdings.

The dashboard handles and displays:

* Stock names and exchange information

* Purchase price, quantity, and total investment

* Portfolio weight allocation

* Present value computations

* Gain/Loss evaluations

* Sector-wise grouping with consolidated summaries

The design emphasizes clarity, accuracy, and usability. Drishti aims to make portfolio analysis straightforward by presenting key insights in a clean layout that supports quick interpretation and decision-making.
